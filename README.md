# CAMRS.--A-Camera
This is a camera modeled by myself, with a custom designed pcb, and optimized system for cost-effectiveness which allows for this camera to be made within a lower price. Allowing for the user to also tweak easily and be a system that takes clear cut pictures efficiently. 

Description of the Project:

The CAMRS. Camera is a Camera that uses a Raspberry PI 3 Model B as a central control module paired with a custom designed PCB that allows for an efficient and smooth camera experience. With a side opening to allow for the user to gain access to all of the electronics inside the user can be able to easily tweak any of the electronics inside.

The CAMRS. Camera is a camera that allows for the user to have a cheap camera that efficiently works for the user, through the usage of easily tweakable electronics. With using easily changeable lens modules, much cheaper than normal lens's this allows for a much cheaper lens allocation compared to normal camera lens's. The main goal is to cater to the user while allowing for the best experience while also allowing for an efficient system for the user. 

**Why this Project?**
The CAMRS. project is a project I wanted to start after I discovered my love for cameras. After looking into the hobby of photography for a few years I got into actually wanting to making cameras myself. I found myself with a camera I couldnt fix when it did break and I wanted a camera I could rely on and also a camera that could be something that I made and I could tweak at any moment and is assecible to the user themselves. I wanted to make a camera that I made, and was something that was cheap and efficient for myself. This is why I made this project.
